About This Project

This project is a web application that simulates the three-body problem and the two-body problem. The three-body problem is a famous problem in physics and mathematics that describes the motion of three celestial bodies under their mutual gravitational attraction. The two-body problem is a simpler version of the three-body problem that describes the motion of two celestial bodies under their mutual gravitational attraction.

The simulations are created using Python and the Flask web framework. The videos are generated using Matplotlib and FFmpeg. The user can input the initial positions, velocities, and masses of the bodies to see how they evolve over a time of 10 seconds.

This project is intended for educational purposes and to provide an interactive way for people to learn about the three-body problem and the two-body problem.

Created by Rhema Wen and Vincent Ji for the Westminster School C2 Project, 2026
